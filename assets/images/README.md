## Como usar:
1. Exporte as imagens do Figma (PNG/JPG)
2. Coloque nesta pasta `assets/images/`
3. As URLs no código irão funcionar automaticamente

## URLs no código:
```css
background-image: url('./assets/images/hero-woman-yellow.jpg');
```
